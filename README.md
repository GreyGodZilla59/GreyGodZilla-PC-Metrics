Portable all-in-one Windows PC monitor — single `.exe`, no installer, no account.

## Quick start

1. Download and unzip anywhere (Desktop, `C:\Tools`, etc.)
2. Run `Grey GodZilla PC Metrics.exe`
3. Click **🧠 AI** in the header or open **Insights** for live stutter diagnosis
4. Right-click the tray icon for overlay, benchmark, fans, and more

Windows SmartScreen may warn on first run — normal for unsigned indie apps.

## Features

### AI Stutter Advisor (v1.0.4)
- Live verdict: GPU bound, CPU bound, thermal, RAM pressure, and more
- Actionable fixes in plain English
- **Copy for Discord** — paste-ready diagnostic bundle
- **Save HTML report** — shareable stutter report

### Live monitoring
- CPU, GPU, RAM, VRAM, temps, power, disk I/O, network
- Custom in-game overlay + 7 themes
- Real-time FPS + frame-time stats (PresentMon bundled)

### Tools
- Standalone benchmark with Score Book
- Fan curves, SMART health, WHEA scan, ping monitor

## Command line

```bat
"Grey GodZilla PC Metrics.exe" --benchmark --quick
"Grey GodZilla PC Metrics.exe" --self-test
```
